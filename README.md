# TDD App template
Simple TDD app template (for sketching out ideas)

## Getting started
1. Clone the repo
2. Run `yarn` to install dependencies

## Tech stack
- Node.js
- TypeScript (ts-node)
- Jest
- EsLint
- Prettier
- Nodemon

## Available scripts
`yarn test` - run tests

`yarn start` - execute the app

`yarn dev` - run the app in development mode with nodemon
